Getting Started

Up and Running in Two Steps

Everything you need to start building beautiful pages

Install

Step 1 of 2
  1. 1
    Clone the repo or install via npm: npm install active_canvas
  2. 2
    Run the local server: npm run dev
  3. 3
    Connect your AI key (OpenAI, Anthropic, or Gemini) in .env

Keep your API keys private. The app runs locally and never uploads your content.

Create a Page

Step 2 of 2
  • Start from a template or blank canvas
  • Edit visually, then open code view to adjust classes
  • Use AI prompts to translate a screenshot into HTML

Accessibility tip: keep headings ordered and add aria-label to interactive elements.

CLI Snippets

terminal 
# new project
npx create-active-canvas my-site

# start dev server
npm run dev

# build for production
npm run build

# export static HTML
npm run export

Commands are illustrative; adjust to your tooling. Output is plain HTML/CSS.
