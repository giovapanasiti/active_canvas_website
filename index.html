Core Features

An incredible tool for web professionals. A visual page builder, merged with an HTML editor, freeing your creative soul.

100% Frontend Editing

Complete "direct manipulation" editing experience. Content editing is quick via point and click on text, images, icons, embeds.

Live HTML Editor

Full access to your code. Edit HTML/CSS directly with syntax highlighting. No proprietary formats, no vendor lock-in.

Zero Bloat Performance

No extra JS or CSS added. Achieve top PageSpeed results with our ultra-lightweight architecture.

Pre-built Sections

Start fast with readymade sections and themes. A well-harmonized set of primitive sections you can combine and remix.

Responsive Preview

Integrated responsive preview tool. Test your designs on any device size in real-time without leaving the editor.

Export as Static HTML

Export pages to static, independent HTML files. Deploy anywhere — Vercel, Netlify, or your own server.
